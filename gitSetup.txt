Create a new repository https://github.com
-Do not initialize with a README

Copy the repository URL

INITIAL
>
git init
git remote add origin https://github.com/3826/RepoName.git
git add .
git commit -m "Initial commit"
git branch -M main
git push -u origin main

UDATE
>
git add . || git add path/to/file.js
git commit -m ""
git push


IF fail to push, sync with remote:
>
git pull origin main --rebase
